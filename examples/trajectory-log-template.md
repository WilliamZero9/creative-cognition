# Creative Trajectory Log

Decision records capturing what was tried, what worked, what didn't, and why.

## Format

Each entry follows this structure:
- **Date**: YYYY-MM-DD
- **Task type**: (naming / ideation / architecture / copy / design / meta-creative)
- **Approach**: lenses, constraints, stolen fire methods, or other techniques used
- **Outcome**: landed / partial / missed
- **Signal**: what indicated the outcome (explicit praise, riff, flat response, rejection, etc.)
- **Principle extracted**: the transferable insight (essence form)
